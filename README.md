# Better-yt-playlists

a better way of making, using and displaying playlists on youtube!

## Development

  `npm install`

  `web-ext lint`

  `web-ext run`

### Options

Run in a specific Firefox instance

    `--firefox=[path/to/ firefox.exe (windows) / firefox-bin (unix)`

Verbose Mode (for debugging?)

      `--verbose`

## Build

 `web-ext build`
